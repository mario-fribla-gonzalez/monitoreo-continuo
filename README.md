# autoMonitoreo Continuo.

Este repositorio posee YAML que nos permiten relizar las siguientes Monitoreos:

- 001-limpieza-historia.yml: nos permite limpiar el historial de un respositorio GIT.

- 888-ping.yml: Nos permite monitorear los Runner de la Organizacion o Usuario GitHub. Informa por email si la algun runner esta fuera de servicio.

- 889-githubstatus.yml: Nos permite monitorear el status de la plataforma GitHub. Informa por email si algun servicio de GitHub esta con problemas.

- 777-runping.yml: Woekflow Schedule para ejecutarse continuamente para el monitoreo de los Runner y Servicio GitHub.

- 999-Limpia-Action-Run-Workflow.yml: Permite eliminar de un repositorio las ejecuciones de workflow realizadas.
